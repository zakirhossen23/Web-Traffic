Overview

Surfy is a powerful & responsive Traffic Exchange System built based on the standard Laraval, an open-source PHP web framework that is largely based on the MVC architecture with many great features, user friendly and easy to customize.

You simply install, add packages, set up the payment gateway, and you are ready to start your own Traffic Exchange Website!

 

Installation

1. Server Requirements

Surfy requires certain technical requirements to be met for the website to operate properly:

  1. PHP version between 5.5.9 - 7.1.*
  2. OpenSSL PHP Extension
  3. PDO PHP Extension
  4. Mbstring PHP Extension
  5. Tokenizer PHP Extension
  6. JSON PHP Extension
  7. Rewrite Module (Apache or Nginx)

File and folder permissions

  1. /bootstrap       775
  2. /storage          775 (recursively)

These extensions must be enabled for Surfy to install properly on the web server.

 

2. Installing Surfy

 

Step 1 : Extract and Upload

Unzpip the Surfy Traffic Exchange package and upload the contents of Surfy upload folder to your server root, usually will be one of the following paths :
- /path/to/www/
- /path/to/html/
- /path/to/public_html/
- /path/to/www/vhosts/domain.tld/httpdocs/

Important : Always use a desktop application like FileZilla to upload files to FTP as opposed to web based file managers to avoid issues with corrupted or not properly uploaded files.

 

Step 2 : Creating Database

Create a database for SurfyApp through your server control panel. If your server has phpMyAdmin, then you can also create a database using phpMyAdmin.

  • Open /.env file.
  • Copy and paste your database details in the file.
CopyDB_DATABASE=homestead
DB_USERNAME=homestead
DB_PASSWORD=secret
  • Save changes and re-upload the /.env file.
  • You are done!

 

Step 3 : Run Installation Wizard

After you created a database and uploaded the files, simply open up your site in the browser (like http://mysite.com, you will redirect to http://mysite.com/install) and follow on-screen instructions to complete the installation.

SMTP

What is SMTP Server ?

An SMTP (Simple Mail Transfer Protocol) server is an application that’s primary purpose is to send, receive, and/or relay outgoing mail between email senders and receivers. Most e-mail systems that send mail over the Internet use SMTP to send messages from one server to another; the messages can then be retrieved with an e-mail client using either POP or IMAP. In addition, SMTP is generally used to send messages from a mail client to a mail server. This is why you need to specify both the POP or IMAP server and the SMTP server when you configure your e-mail application.

NOTE : Some hosting providers blocks port 25 or 465 in that case you have change your SMTP email server or contact your hosting provider.

Setup SMTP Server

Using your own or some SMTP server /.env file setup

  • Login to your FTP account.
  • Open and Edit /.env file.
  • Copy and paste your server smtp details in the file.
CopyMAIL_DRIVER=smtp
MAIL_HOST=your-mail-host
MAIL_PORT=your-mail-port
MAIL_USERNAME=your-mail-username
MAIL_PASSWORD=your-mail-password
MAIL_FROM_ADDRESS=your-mail-address
MAIL_FROM_NAME=your-mail-name
MAIL_ENCRYPTION=your-mail-encryption
  • Replace your-mail-host, your-mail-port, your-mail-username, your-mail-password, your-mail-address, your-mail-name and your-mail-encryption.
  • Save changes and re-upload the /.env file.
  • You are done!

Captcha

Recaptcha Key

What is recaptcha?

reCAPTCHA is a free CAPTCHA service that helps protect your site against spam, malicious registrations and other forms of attacks where computers try to disguise themselves as a human. reCAPTCHA comes in the form of a widget that you can easily add to your blog, forum, registration.

How to setup.

  • Visit https://www.google.com/recaptcha/admin
  • Follow the steps by select the reCAPTCHA version and get your site key and your secret key.

/.env file setup

  • Login to your FTP account
  • Open /.env file.
  • Provide the keys there.
CopyNOCAPTCHA_SECRET=your-recaptcha-secret_key
NOCAPTCHA_SITEKEY=your-recaptcha-site_key
  • Replace  your-nocaptcha-site_key and your-nocaptcha-secret_key by your keys.
  • Save the file.
  • You are done.

Cron Job

How can I start the Scheduler?

The Scheduler will execute:

  • The Credits Clear command (Set credits to 0 where credits are less than 0) every hour.

When using the scheduler, you only need to add the following Cron entry to your server.

Copy* * * * * {PHP_BIN_PATH} /path/to/YourWebsite/artisan schedule:run >> /dev/null 2>&1

This Cron will call the Laravel command scheduler every minute. When the schedule:run command is executed, Laravel will evaluate your scheduled tasks and runs the tasks that are due.

PHP command examples:

General example:

  • /usr/local/bin/php /home/user/public_html/artisan schedule:run >> /dev/null 2>&1

Domain-specific example:

  • /usr/local/bin/ea-php99 /home/user/yoursite.com/artisan schedule:run >> /dev/null 2>&1
NOTE : In the above example, replace "ea-php99" with the PHP version assigned to the domain you wish to use. Look in the MultiPHP Manager from the cPanel for the actual PHP version assigned to a domain.

More information about Cron Job:

  • https://crontab.guru/every-1-hour
  • https://code.tutsplus.com/tutorials/scheduling-tasks-with-cron-jobs--net-8800
  • http://askubuntu.com/questions/2368/how-do-i-set-up-a-cron-job

How To

How can I setup the packages/Plans?
  • Go to your Admin panel -> Packages -> Packages List
  • Select a package and edit it
  • NOTE: You can add new packages or delete current packages in the actual version.
How can I change the default Autosurf time?
  • Go to your Admin panel -> System -> General Settings -> Surf Tab
  • In the next field, type the number of seconds which will be used to display each website during the autosurf exchange.
  • Click Save Changes.
How to manually add a new user /or admin accounts?
  • Go to your Admin panel -> Users -> Add User
  • Fill out the forms with information about the new user.
  • Click Save Changes.

Please see more features by visiting your Admin panel -> System -> General Settings...
